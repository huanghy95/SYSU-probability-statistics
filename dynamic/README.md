# SYSU-probability-Dynamemic_zero

**Please run in Linux**

## To run in bash

```
cd ./release
chmod +x ziptxt unziptxt
```

### TO zip

```
./ziptxt <input> <output>
```

### TO unzip

```
./unziptxt <input> <output>
```

## To run in python(3)

```
cd ./code
pip install -r requirements.txt
```

### TO zip

```
python ziptxt.py <input> <output>
```

### TO unzip

```
python unziptxt.py <input> <output>
```

## TO check the difference

```
diff <file1> <file2>
```
